Welcome to Legend of Legends Base Stats program!

This program has a table of all the base statistics of all the champions within League of Legends

The data for this table is taken from the web each time this program is open, ensuring it is up to date

To get started click on either the Champion Stats button for their base statistics or the Champion Comparison button to compare
